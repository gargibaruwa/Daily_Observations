Instructions to run the application:
1. Connect to remote server and login with unity ID and password.
2. Login to Oracle DB using appropriate credentials.
3. Run the file gbaruwa_DDLs.sql in the oracle database.
4. Run the file .mycshrc using the command “source .mycshrc”.
5. Run the file gbaruwa.java in the unix box using the commands
	javac  gbaruwa.java
	java gbaruwa
